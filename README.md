# JovialityApp
这是一款综合类app,参照的是一款名为“简阅”的app(非常感谢)
暂时划分为3个模块
（1）新闻阅读
（2）图片浏览
（3）视频观看

目前正处于开发的初级阶段，慢慢了解如何构建一个好的项目。
在学习别人的项目的时候，才发现自己以前写的是有多烂（层次结构，封装，扩展，耦合度，依赖库等都存在问题）
虽然也可以做出一个功能基本完善的产品，但是维护起来非常困难，扩展性非常差。
